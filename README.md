# IMDB-Review-Sentiment-Analysis

This project builds a sentiment classifier on IMDb TV show reviews to identify titles generating the strongest positive and negative reactions.

## Approach
- TF-IDF text features
- Linear SVM / Logistic Regression
- Binary sentiment classification
- ROC + confusion matrix evaluation

## Results
After refining labels to focus on extreme ratings, the model achieved:
- ROC AUC: 0.94
- Strong separation between highly positive and negative reviews

## Why it matters
This can help prioritize shows with the most polarized audience sentiment, supporting marketing, content, and retention decisions.
